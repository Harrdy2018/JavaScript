# 定时器的使用



## 开启定时器
* Ex.1 setInterval() 间隔型
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
<script>
    function show() {
        alert('a');
    }
    //每间隔1秒钟，弹出一个a
    setInterval(show,1000);
</script>
</head>
<body>

</body>
</html>
```

***
* Ex.2 setTimeout() 延时型
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
<script>
    function show() {
        alert('a');
    }
    //只执行一次
    setTimeout(show,2000);
</script>
</head>
<body>

</body>
</html>
```

***
